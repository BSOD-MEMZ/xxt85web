项目详情

了解 QQListener 的运行机制与配置方法

详细工作原理

QQListener 灵活支持多种环境，确保消息抓取的稳定性：

  • WinSDK 模式： 通过监听 Windows 通知中心（Toast）实现。这是最佳实践，功耗极低，几乎不占资源，墙裂推荐。
  • UIA 模式： 备选方案。针对不支持通知的老旧环境，通过 UI 自动化技术直接从窗口元素中“扒”消息。

快速上手指南

  1. 版本限制： 必须使用 NT QQ。老版 QQ 不发 Windows 通知。系统要求 Win10 1903+。
    下载新版 QQ
  2. 权限申请： 在 Windows 设置里把 QQ 的“在操作中心显示通知”打开。
  3. 路径配置： 在设置里找到聊天记录保存位置，同步填入 QQListener 的保存目录。
