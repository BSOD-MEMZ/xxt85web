详细工作原理

QQListener 提供了两种工作模式：

  • WinSDK 模式： 通过监听 Windows 通知中心（Toast）实现。功耗极低，推荐使用。
  • UIA 模式： 针对不支持通知的老旧环境，通过 UI 自动化技术直接抓取窗口元素。

使用方法

  1. 版本限制：本程序只支持NT QQ，因为旧版QQ根本没法发通知下载新版QQ。本程序只能在Windows 10 1903及以上版本工作
  2. 软件安装：将 QQListener 安装到你想要的位置
  3. 权限申请：请务必在 Windows 设置中开启 QQ 的“在操作中心显示通知”，并且允许 QQ 发送 Windows 通知
  4. 路径配置：在设置中找到聊天记录保存位置，然后填入 QQListener 的聊天记录保存文件夹
