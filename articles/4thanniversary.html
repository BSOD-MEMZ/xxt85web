xxtsoft 四周年

一个小站 · 四年光阴
2022 — 2026

草，这个网站居然活了四年了。

四年前的这个时候，我还是个连 HTML 是啥都不知道的小屁孩，靠着朋友帮忙搭了个 WordPress 就算"有网站"了。那时候觉得，哇，我在互联网上有自己的地盘了，虽然那个地盘基本都是别人帮我砌的墙。

四年后，我坐在这里，用着 VSCode 一个字一个字地敲下这些文字。旁边是那台从404教室里"解放"出来的电脑，Win11 的云母效果在 4K 屏幕上流光溢彩。我甚至有点恍惚——我居然真的学会了怎么写网页。

四年，五个网站

如果你去看博物馆，你会发现我搞过五个版本的网站。第一个是 WordPress 套皮，第二个第三个用了 Hexo，第四个是 Halo，第五个——也就是你现在看到的——是纯手搓的静态 HTML。

说实话，每次换网站都像是一次重生。旧的域名过期了，旧的截图找不到了，旧的记忆也模糊了。但每次重新开始，我都比上一次更懂一点。第一个网站我连 FTP 都不会用，现在我已经能自己写 CSS、搞响应式、配 Git 了。

这大概就是成长的证据吧。

这个网站对我意味着什么

废话，当然是我的赛博垃圾桶啊。

在学校被老师压榨，回家被作业淹没，唯一能让我喘口气的地方就是这里。我可以在这里骂学校、骂应试教育、骂一切我想骂的东西，不用担心被叫去年级组办公室喝茶。我可以在这里分享我写的破程序，虽然没几个人用，但自己看着爽就行。

更重要的是，这个网站是我存在过的证明。若干年后，也许没人记得 xxt8582753 是谁，但只要这个域名还在、这些 HTML 文件还在，就有人知道我曾在互联网上留下过痕迹。

数据很难看，但我很开心

四年了，本站的访问量还不如别人一篇 CSDN 水文的多。必应搜索量还没我 GitHub 提交次数多。但我一点都不在乎——好吧，其实有一点点在乎。

因为这不是一个商业网站，没有 SEO 优化，没有广告，没有引流。它就是一个高中生的个人空间，用着 Vista 风格的 UI，写着一些别人看不懂的碎碎念。如果有人偶然点进来，看到这些文字，笑了或者怒了，那这个网站就值了。

而且你知道吗，虽然数据惨淡，但我确实收到过一些陌生人的反馈。有人说我的程序帮到了他，有人说看了我的文章觉得有意思。这些零星的正面反馈，比一万个 AI 洗稿网站的流量都有意义。

感谢名单

四年了，这个网站不是我一个人撑起来的。

  • AbCd — 我最好的朋友，帮我买域名、搭服务器、写代码、在404教室放风。没有你就没有这个网站。
  • LAGSNES — 提供了最早的服务器，让我的网站能在互联网上生根，而不是孤独地躺在本地磁盘里。
  • SK — 在本站投稿了 SKToolBox，让我知道这个网站也能成为别人分享的平台。
  • 所有在B站关注我的人 — 虽然视频质量不咋地，但你们的点赞和评论是我继续下去的动力。
  • 每一个点进来看过的人 — 感谢你在这个信息爆炸的时代，愿意花时间看一个高中生的碎碎念。

四年来的变化

四年前我还在搞 Scratch，现在已经能写 PyQt 桌面应用了。

四年前我用的是那台开机三分钟的破笔记本，现在我有了一台 12 代 i5。

四年前我只会用 WordPress 模板，现在我能手搓整个网站。

四年前我在初中混日子，现在我在高中继续混日子。

但最大的变化大概是——我开始有自己的想法了。四年前我只会跟着别人走，现在我知道自己想要什么、想做什么、想成为什么样的人。虽然前路依然迷茫，但至少我不再是那个只会复制粘贴代码的小白了。

未来？

说实话，我不知道这个网站还能活多久。

域名需要续费，高中学业越来越重，大学的门槛也越来越高。也许哪一天我累了，不想再更新了，这个网站就会变成数字废墟，静静地躺在 Netlify 的服务器上，等待域名过期的那一天。

但在那之前，我会继续写下去。

继续写代码，继续写文章，继续骂学校，继续分享我觉得有意思的东西。因为这个网站是我在压抑的高中生活中唯一的自由之地，是我向这个世界喊话的扩音器——虽然音量不大，但至少我在喊。

四年了，xxtsoft 还在。

希望下一个四年，我们还在这里。
